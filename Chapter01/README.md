### Membuat kode program
 
1. Buat kode program fungsi : c = a + b
2. Buat kode program fungsi z = Wx dimana W adalah matriks n kali n. x adalah vektor ukuran n. dengan n = 2.1957
3. Buat kode program fungsi z = Wx + b
